Тестовое задание для компании е.Ком
Проект содержит Dockerfile, его можно запустить двумя разными способами:
1. Через терминал вводится команда: uvicorn main:app --reload , соответственно в директории, 
где находится исполнительный файл.
2. С помощью Docker нужно ввести две команды в терминале:
    docker build . -t fastapi_app:latest
    docker run -p 7777:8000 fastapi_app
Порт выбирал 7777, но можно какой вам удобно, так же в теории можно было выложить этот образ на docker hub,
но я этого делать не стал.
Для запуска в docker используется другая утилита - gunicorn.
Для тестов изначально использовал словарь templates, он находится в файле models.py. 
Скрипт для тестирования в фале test.py.
Версия python:3.9.13
